WP Framework is a blank WordPress theme framework, which aims to cut down on your theme development time.
It gives you a solid theme foundation to build your project on top of so you can focus on project-specific features right from the git-go.

To Learn more about how you can take advantage of this amazing Framework,
visit the website for more information <http://wpframework.com>.

Installation
------------

Manual Install:
To download a copy of WP Framework, visit <www.wpframework.com> and download the zip archive.
Extract the archive and Upload the /wpframework/ folder into your 'wp-content/themes' directory.

1.  Login to your WordPress Admin and navigate to the Apperance Tab
2.  Select WP Framework then Activate it.
3.  That's it! Get started building your theme!

SVN Style:
To download a copy of WP Framework, navigate into your 'wp-content/themes' directory, then copy & paste
the following lines in your subversion client:

svn co http://wpframework.googlecode.com/svn/trunk wpframework

1.  Login to your WordPress Admin and navigate to the Apperance Tab
2.  Select WP Framework then Activate it.
3.  That's it! Get started building your theme!
